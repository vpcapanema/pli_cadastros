Recursos do Sistema

Desenvolvido com as melhores práticas de segurança e usabilidade para atender todas as necessidades de gerenciamento de cadastros.

Pessoa Física

Cadastro completo com validação de CPF, endereço integrado com ViaCEP e campos personalizáveis.

Pessoa Jurídica

Gestão empresarial com validação de CNPJ, consulta automática de dados e histórico de alterações.

Usuários

Controle de acesso com níveis de permissão, autenticação segura e recuperação de senha.

Segurança

Criptografia de dados, autenticação JWT, proteção CSRF e auditoria completa de ações.

Pesquisa Avançada

Filtros inteligentes, busca em tempo real e exportação de dados em múltiplos formatos.

Responsivo

Interface adaptável para desktop, tablet e smartphone com experiência otimizada em todos os dispositivos.

Sobre o Sistema

Desenvolvido especificamente para atender às necessidades do PLI (SIGMA - Sistema Integrado de Gerenciamento, Monitoramento e Análise de Informações do Plano de Logística e Investimento do Estado de São Paulo), este módulo oferece uma solução completa para gerenciamento de cadastros.
